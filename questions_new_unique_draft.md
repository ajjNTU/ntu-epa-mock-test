## 01 Introductory Maths

1. Q: Solve `9x + 3 = -2`.
   A: `x = -5/9`.

2. Q: Which logarithm rule rewrites `log(a/b)`?
   A: `log(a) - log(b)`.

3. Q: A right-angled triangle has sides 5 and 12. What is the hypotenuse?
   A: `13`.

## 02 Introduction to Programming

1. Q: Which Python value represents the absence of a value?
   A: `None`.

2. Q: Which OOP principle allows different classes to use the same method name with different behaviour?
   A: `Polymorphism`.

3. Q: Which Python error is raised when you access a list position that does not exist?
   A: `IndexError`.

## 03 Foundations of Data Science

1. Q: In the six-phase lifecycle, which phase is mainly about choosing methods, features, metrics, and evaluation strategy?
   A: `Model Planning`.

2. Q: JSON and XML are best described as what kind of data?
   A: `Semi-structured data`.

3. Q: In association rules, what does confidence measure?
   A: `How often the consequent occurs when the antecedent occurs`.

## 05 Systems Analysis and Design

1. Q: In a use case diagram, which relationship shows optional behaviour added only in certain conditions?
   A: `extend`.

2. Q: Which requirements-gathering technique is best for seeing how users actually perform their work?
   A: `Observation`.

3. Q: Which client architecture keeps most application logic on the server side?
   A: `Thin client`.

## 06 LSEP in Data Science

1. Q: Which GDPR/DPA principle requires personal data to be correct and kept up to date?
   A: `Accuracy`.

2. Q: Which type of intellectual property mainly protects names, logos, and brand identifiers?
   A: `Trademark`.

3. Q: What is the main problem with a predatory journal?
   A: `It presents itself as scholarly without credible peer review or editorial standards`.

## 07 Advanced Systems Programming

1. Q: In RUP, which phase is concerned with moving the product to its users?
   A: `Transition`.

2. Q: Which type of cohesion is considered the best because all elements support one clear responsibility?
   A: `Functional cohesion`.

3. Q: What type of coupling exists when modules share global data?
   A: `Common coupling`.

## 08 Interpreting Data

1. Q: What is a Type II error?
   A: `Failing to reject a false null hypothesis`.

2. Q: Which plot is usually best for showing the relationship between two numeric variables?
   A: `Scatter plot`.

3. Q: In PCA, what does a loading plot show?
   A: `How strongly variables contribute to the principal components`.

## 09 Project Management

1. Q: In the module’s project types, what is a project called when both the goal and the process are uncertain?
   A: `A foggy project`.

2. Q: In schedule networks, what does float or slack represent?
   A: `How much an activity can slip without causing schedule problems`.

3. Q: Which process group is responsible for formal project completion?
   A: `Closing`.

## 10 Machine Learning

1. Q: Which CRISP-DM phase checks whether the model actually meets the business or research objective?
   A: `Evaluation`.

2. Q: Which scaling method transforms features so they have mean 0 and standard deviation 1?
   A: `Z-score standardization`.

3. Q: Why is it bad practice to keep using the test set while tuning a model?
   A: `It leaks evaluation information and gives an optimistically biased performance estimate`.

## 11 Information Security

1. Q: Which part of the CIA triad is about information remaining accurate and unaltered?
   A: `Integrity`.

2. Q: What is the difference between authentication and authorisation?
   A: `Authentication proves identity; authorisation determines permissions`.

3. Q: What is the main purpose of an asset register?
   A: `To document assets, ownership, value, and risk-related details for later analysis`.

## 12 Scalable Systems and Databases

1. Q: In Hadoop, what does data locality mean?
   A: `Moving computation close to where the data is stored`.

2. Q: What does BASE emphasise in many NoSQL systems?
   A: `Basically Available, Soft state, Eventually consistent behaviour`.

3. Q: Which Hadoop ecosystem tool is better suited to ingesting streaming logs/events than bulk RDBMS transfer?
   A: `Flume`.

## 13 AI for Data Science

1. Q: Which agent type chooses among alternatives using a utility function?
   A: `Utility-based agent`.

2. Q: What is an admissible heuristic?
   A: `A heuristic that never overestimates the true remaining cost`.

3. Q: In NLP, what is the main limitation of a Bag-of-Words representation?
   A: `It ignores word order and syntax`.

## 14 Project Planning

1. Q: Why is stakeholder analysis important during planning?
   A: `It helps identify requirements, constraints, acceptance issues, and organisational feasibility`.

2. Q: How does a goal differ from an objective?
   A: `A goal is broad; an objective is a specific measurable step toward it`.

3. Q: Why is CRISP-DM not enough on its own as a full project-management method?
   A: `It structures analytics work well but is weak on coordination, roles, and broader project management`.
